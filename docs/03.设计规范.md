## 📐 布局规范

### 容器宽度

```
max-width: 1280px (max-w-7xl)  - 内容容器最大宽度
max-width: 1024px (max-w-5xl)  - 文章阅读容器
max-width: 768px  (max-w-3xl)  - 文章正文
```

### 间距系统

使用 Tailwind 的 spacing scale:

```
xs:  4px   (1)
sm:  8px   (2)
md:  12px  (3)
lg:  16px  (4)
xl:  24px  (6)
2xl: 32px  (8)
3xl: 48px  (12)
4xl: 64px  (16)
5xl: 96px  (24)
```

**应用场景**:
- 卡片内边距: `p-6` 或 `p-8`
- 区块间距: `mb-12` 或 `mb-16`
- 列间距: `gap-6` 或 `gap-8`

### 网格布局

```css
/* 精选文章 - 2列 */
.featured-grid {
  display: grid;
  grid-template-columns: repeat(2, 1fr);
  gap: 2rem;
}

/* 文章列表 - 3列 */
.article-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 1.5rem;
}

/* 响应式 */
@media (max-width: 768px) {
  .featured-grid,
  .article-grid {
    grid-template-columns: 1fr;
  }
}
```

## 🎭 动画和过渡

### 悬停效果

```css
/* 卡片悬停 */
.card {
  transition: all 0.3s ease;
}
.card:hover {
  box-shadow: 0 20px 25px -5px rgb(0 0 0 / 0.1);
  border-color: #93C5FD;
}

/* 标题悬停 */
.title {
  transition: color 0.2s ease;
}
.title:hover {
  color: #3B82F6;
}

/* 按钮悬停 */
.button {
  transition: background-color 0.2s ease;
}
```

### 页面过渡

```css
/* 淡入动画 */
@keyframes fadeIn {
  from {
    opacity: 0;
    transform: translateY(10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.fade-in {
  animation: fadeIn 0.5s ease-out;
}
```

## ✅ 设计检查清单

### 视觉一致性

- [ ] 所有标题使用统一的字号和字重
- [ ] 所有卡片使用统一的圆角和阴影
- [ ] 所有交互元素有统一的悬停效果
- [ ] 颜色使用保持一致（不超过 3-4 种主色）

### 可读性

- [ ] 文字和背景有足够对比度（至少 4.5:1）
- [ ] 行高舒适（正文至少 1.5）
- [ ] 段落间距合理
- [ ] 移动端字号不小于 14px

### 响应式

- [ ] 所有页面在移动端可正常浏览
- [ ] 触摸目标至少 44x44px
- [ ] 移动端没有横向滚动
- [ ] 图片和媒体自适应

### 性能

- [ ] 动画流畅，不卡顿
- [ ] 图片优化，加载快速
- [ ] 首屏渲染时间 < 1.5s
- [ ] 无不必要的重排重绘

## 🎯 设计原则总结

1. **少即是多** - 去除不必要的装饰，保持简洁
2. **一致性** - 保持视觉和交互的一致性
3. **层次感** - 通过大小、颜色、间距建立视觉层次
4. **可用性** - 优先考虑用户体验和可访问性
5. **美观** - 在功能基础上追求视觉美感
