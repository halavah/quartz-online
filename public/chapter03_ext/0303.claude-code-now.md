## 1. 什么是 Claude Code Now？

Claude Code Now 是一款专为 Claude Code 设计的高效启动器，解决了传统命令行启动方式的繁琐问题。通过图形化界面和智能配置管理，让用户能够以最快速度启动和使用 Claude Code。

### 核心价值

- **极速启动**：一键启动，无需记忆复杂命令
- **多API支持**：支持Anthropic官方、智谱GLM、自定义API
- **文件夹启动**：从任意文件夹直接启动Claude Code
- **配置管理**：智能管理多个API配置和项目设置
- **Dock集成**：完美的macOS Dock集成体验

## 4. 安装指南

### 4.1 官方安装方式

#### 方法1：直接下载安装包
1. 访问 [GitHub Releases](https://github.com/orange2ai/claude-code-now/releases)
2. 下载最新的 `Claude Code Now.app.zip`
3. 解压并将应用拖拽到 Applications 文件夹

#### 方法2：Git克隆安装
```bash
# 克隆仓库
git clone https://github.com/orange2ai/claude-code-now.git

# 进入项目目录
cd claude-code-now

# 复制应用到Applications文件夹
cp -r "Claude Code Now.app" /Applications/
```

#### 方法3：使用安装脚本
```bash
# 下载并运行一键安装脚本
curl -fsSL https://raw.githubusercontent.com/orange2ai/claude-code-now/main/install.sh | bash

# 或者手动下载脚本执行
wget https://raw.githubusercontent.com/orange2ai/claude-code-now/main/install.sh
chmod +x install.sh
./install.sh
```

### 4.2 验证安装

1. **检查应用是否正常安装**
   ```bash
   ls /Applications/"Claude Code Now.app"
   ```

2. **启动应用验证**
   ```bash
   open "/Applications/Claude Code Now.app"
   ```

3. **检查版本信息**
   - 在应用中查看关于页面
   - 或通过命令行检查版本

### 4.3 首次配置

1. **启动Claude Code Now**
2. **允许系统权限**（首次启动时会提示）
3. **配置Claude Code路径**
4. **添加API配置**
5. **设置默认工作目录**

## 5. 配置说明

### 5.1 基础配置

#### Claude Code 路径配置
```json
{
  "claude_code_path": "/usr/local/bin/claude",
  "auto_detect_path": true,
  "default_shell": "/bin/zsh"
}
```

#### API配置示例
```json
{
  "api_configs": {
    "anthropic": {
      "name": "Anthropic Official",
      "api_key": "your-api-key-here",
      "base_url": "https://api.anthropic.com",
      "model": "claude-3-opus-20240229",
      "max_tokens": 4096
    },
    "zhipuai": {
      "name": "智谱GLM",
      "api_key": "your-zhipu-key",
      "base_url": "https://open.bigmodel.cn/api/paas/v4",
      "model": "glm-4",
      "max_tokens": 4096
    }
  }
}
```

### 5.2 高级配置

#### 启动选项配置
```json
{
  "launch_options": {
    "auto_open_terminal": true,
    "default_directory": "~/projects",
    "remember_last_directory": true,
    "startup_animation": true,
    "check_updates": true
  }
}
```

#### 界面配置
```json
{
  "ui_config": {
    "theme": "auto",
    "dock_icon_style": "modern",
    "show_status_in_menu_bar": true,
    "notification_enabled": true
  }
}
```

### 5.3 配置文件位置

- **macOS配置文件**：`~/Library/Application Support/Claude Code Now/config.json`
- **日志文件**：`~/Library/Logs/Claude Code Now/`
- **缓存目录**：`~/Library/Caches/Claude Code Now/`

## 7. API配置管理

### 7.1 支持的API类型

#### Anthropic Claude API
```json
{
  "type": "anthropic",
  "name": "Claude Official",
  "api_key": "sk-ant-xxxxx",
  "base_url": "https://api.anthropic.com",
  "models": ["claude-3-opus-20240229", "claude-3-sonnet-20240229", "claude-3-haiku-20240307"]
}
```

#### 智谱GLM API
```json
{
  "type": "zhipuai",
  "name": "智谱GLM-4",
  "api_key": "xxxxxxxx",
  "base_url": "https://open.bigmodel.cn/api/paas/v4",
  "models": ["glm-4", "glm-3-turbo"]
}
```

#### 自定义API
```json
{
  "type": "custom",
  "name": "自定义服务",
  "api_key": "your-key",
  "base_url": "https://your-api.com/v1",
  "headers": {
    "Custom-Header": "value"
  },
  "models": ["custom-model"]
}
```

### 7.2 API配置最佳实践

1. **密钥安全**
   - 不要在配置文件中硬编码API密钥
   - 使用环境变量存储敏感信息
   - 定期轮换API密钥

2. **性能优化**
   - 选择合适的模型
   - 设置合理的token限制
   - 配置超时时间

3. **成本控制**
   - 监控API使用量
   - 设置使用限制
   - 选择性价比高的模型

## 8. 故障排除

### 常见问题

#### 1. 应用无法启动
```bash
# 检查系统权限
sudo spctl --assess --verbose "/Applications/Claude Code Now.app"

# 重新签名应用（如果需要）
codesign --force --sign - "/Applications/Claude Code Now.app"
```

#### 2. Claude Code路径错误
```bash
# 查找Claude Code路径
which claude

# 手动设置路径
export PATH="/usr/local/bin:$PATH"

# 验证安装
claude --version
```

#### 3. API连接失败
- 检查网络连接
- 验证API密钥正确性
- 确认API端点可用性
- 查看防火墙设置

#### 4. Dock图标问题
```bash
# 重置Dock配置
defaults delete com.apple.dock
killall Dock

# 重新添加应用到Dock
```

### 调试模式

#### 启用调试日志
1. **打开终端**
   ```bash
   # 启用调试模式
   defaults write com.orange2ai.claude-code-now debugMode -bool true
   ```

2. **查看日志**
   ```bash
   # 查看应用日志
   tail -f ~/Library/Logs/Claude\ Code\ Now/app.log

   # 查看系统日志
   log show --predicate 'process == "Claude Code Now"'
   ```

3. **收集诊断信息**
   ```bash
   # 生成诊断报告
   "/Applications/Claude Code Now.app/Contents/MacOS/claude-code-now" --diagnostic
   ```

### 性能优化

#### 启动速度优化
1. **清理缓存**
   ```bash
   rm -rf ~/Library/Caches/Claude\ Code\ Now/
   ```

2. **优化配置**
   - 减少启动项
   - 简化API配置
   - 禁用不必要的功能

3. **系统优化**
   - 增加虚拟内存
   - 优化SSD性能
   - 关闭后台应用
