Claude Code 安装教程

详细的图文指引，帮助你在 Windows、MacOS、Linux 平台快速安装和配置 Claude Code AI 编程助手

项目介绍

Claude Code 是 Anthropic 推出的命令行 AI 编程助手，它能够直接在终端中帮助你编写代码、调试问题、解释代码逻辑，并自动执行各种开发任务。

与传统的代码编辑器插件不同，Claude Code 是一个独立的命令行工具，具有更强大的上下文理解能力和自主操作能力。它可以：

  • 理解整个项目的代码结构和上下文
  • 自动读取、编辑和创建文件
  • 执行命令行工具和脚本
  • 进行代码审查和重构建议
  • 生成测试用例和文档

AI 模型配置方法

Claude Code 需要 Anthropic API Key 才能正常工作。你可以通过以下几种方式配置：

方法一：环境变量配置（推荐）

在系统环境变量中设置 API Key，这是最常用的方法：

Linux / MacOS 
# 在 ~/.bashrc 或 ~/.zshrc 中添加
export ANTHROPIC_API_KEY="sk-ant-api03-your-api-key-here"
export ANTHROPIC_BASE_URL="https://api.anthropic.com"  # 可选，使用中转服务时需要
Windows PowerShell 
# 临时设置（当前会话有效）
$env:ANTHROPIC_API_KEY="sk-ant-api03-your-api-key-here"

# 永久设置（用户级别）
[Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY", "sk-ant-api03-your-api-key-here", "User")

方法二：配置文件

在 Claude Code 配置目录中创建 settings.json 文件：

~/.claude/settings.json 
{
  "env": {
    "ANTHROPIC_API_KEY": "sk-ant-api03-your-api-key-here",
    "ANTHROPIC_BASE_URL": "https://api.anthropic.com"
  }
}
💡 提示：如果你在中国大陆，可能需要使用 API 中转服务。将 ANTHROPIC_BASE_URL 设置为中转服务的地址即可。

Windows 端图文指引

⚠️ 重要提示：Claude Code 在 Windows 上需要 Git Bash 才能运行。请先确保安装了 Git for Windows。

  1. 安装 Git for Windows

    访问 https://git-scm.com/downloads/win 下载最新版本的 Git for Windows。

    运行安装程序，保持默认选项即可。安装完成后，你将拥有 Git Bash 终端环境。

  2. 使用 PowerShell 安装 Claude Code

    以管理员身份打开 PowerShell，运行以下命令：

    PowerShell 安装命令 
    # 安装最新稳定版本
irm https://claude.ai/install.ps1 | iex

# 或安装指定版本
& ([scriptblock]::Create((irm https://claude.ai/install.ps1))) 1.0.58

  3. 配置环境变量 PATH

    安装完成后，需要将 Claude Code 添加到系统 PATH 中。在 PowerShell 中运行：

    添加到 PATH 
    # 永久添加到用户 PATH
[Environment]::SetEnvironmentVariable("Path", [Environment]::GetEnvironmentVariable("Path", "User") + ";$env:USERPROFILE\.local\bin", "User")

# 立即在当前会话生效
$env:Path += ";$env:USERPROFILE\.local\bin"

  4. 配置 API Key

    设置 Anthropic API Key 环境变量：

    配置 API Key 
    # 永久设置 API Key（替换为你的实际 API Key）
[Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY", "sk-ant-api03-your-key-here", "User")

# 如果使用中转服务，还需要设置 Base URL
[Environment]::SetEnvironmentVariable("ANTHROPIC_BASE_URL", "https://your-proxy-url.com", "User")

  5. 验证安装

    重新打开一个 PowerShell 或 Git Bash 窗口，运行以下命令验证安装：

    验证安装 
    # 查看版本
claude --version

# 测试运行
claude "Hello, Claude!"

# 系统诊断
claude doctor
    ✅ 成功提示：如果能看到版本号和正常响应，说明安装成功！

MacOS 端图文指引

MacOS 上的安装过程最为简单，支持多种安装方式。

  1. 一键安装（推荐）

    打开终端（Terminal），运行以下命令：

    一键安装脚本 
    curl -fsSL https://claude.ai/install.sh | bash

    脚本会自动下载并安装 Claude Code 到 ~/.local/bin 目录。

  2. 使用 Homebrew 安装（可选）

    如果你已经安装了 Homebrew，可以使用以下命令：

    Homebrew 安装 
    brew install --cask claude-code

    使用 Homebrew 的好处是可以通过 brew upgrade 统一管理更新。

  3. 配置 API Key

    在 shell 配置文件中添加环境变量。根据你使用的 shell，编辑对应的配置文件：

    ~/.zshrc 或 ~/.bashrc 
    # 添加以下内容到配置文件末尾
export ANTHROPIC_API_KEY="sk-ant-api03-your-key-here"

# 如果使用中转服务
export ANTHROPIC_BASE_URL="https://your-proxy-url.com"

    保存后，运行 source ~/.zshrc 或 source ~/.bashrc 使配置生效。

  4. 验证安装

    运行以下命令验证安装：

    验证安装 
    # 查看版本
claude --version

# 测试运行
claude "帮我创建一个 Python 脚本"

# 系统诊断
claude doctor

Linux 端图文指引

Linux 系统支持所有主流发行版，包括 Ubuntu、Debian、Fedora、Arch Linux 等。

  1. 一键安装

    在终端中运行安装脚本：

    一键安装脚本 
    curl -fsSL https://claude.ai/install.sh | bash

  2. 检查依赖（Alpine Linux）

    如果你使用的是 Alpine Linux，需要先安装必要的依赖：

    Alpine Linux 依赖 
    # 安装依赖
apk add libgcc libstdc++ ripgrep

# 设置环境变量
export USE_BUILTIN_RIPGREP=0

  3. 配置 API Key

    编辑 shell 配置文件（通常是 ~/.bashrc 或 ~/.profile）：

    ~/.bashrc 
    # 添加以下内容
export ANTHROPIC_API_KEY="sk-ant-api03-your-key-here"
export ANTHROPIC_BASE_URL="https://your-proxy-url.com"  # 可选

    保存后运行 source ~/.bashrc 使配置生效。

  4. 验证安装

    运行验证命令：

    验证安装 
    # 查看版本
claude --version

# 查看安装位置
which claude

# 系统诊断
claude doctor

常见问题

❓ 安装后找不到 claude 命令？

这通常是 PATH 环境变量配置问题。确保 ~/.local/bin（Linux/Mac）或 %USERPROFILE%\.local\bin（Windows）已添加到 PATH 中。

❓ Windows 上提示 "bash: claude: command not found"？

请确保：

  1. 已安装 Git for Windows
  2. 使用 Git Bash 而不是 CMD 或 PowerShell 运行 claude 命令
  3. PATH 环境变量已正确配置

❓ 提示 API Key 无效？

检查以下几点：

  • 确认 API Key 格式正确，以 sk-ant-api03- 开头
  • 如果使用中转服务，确认 ANTHROPIC_BASE_URL 配置正确
  • 重启终端使环境变量生效
  • 运行 claude doctor 检查配置

❓ 如何更新 Claude Code？

Claude Code 支持自动更新，也可以手动触发：

手动更新 
# 检查并安装更新
claude update

# 使用 Homebrew（仅 Mac）
brew upgrade --cask claude-code

❓ 可以在公司内网环境使用吗？

可以，但需要：

  • 确保网络可以访问 Anthropic API（或配置代理）
  • 如果使用企业部署，可以配置 Amazon Bedrock 或 Google Vertex AI
  • 遵守公司的安全策略和代码审查流程
