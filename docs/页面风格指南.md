# Quartz 在线项目 - 页面风格指南

## 项目概述

本文档详细描述了 Quartz 在线项目中现有的各种页面设计风格，为后续的页面开发提供参考和指导。

---

## 🎨 主要页面风格分类

### 1. **深色科技风格** (Dark Tech Style)
**代表文件**: `0101.Xget-超高性能开发者资源加速引擎.html`

#### 设计特点
- **配色方案**:
  - 主色调：`#0066ff` (科技蓝)
  - 次要色：`#00d4ff` (青蓝色)
  - 强调色：`#ff6b00` (橙色)
  - 背景色：`#0a0e27` (深蓝黑)
  - 卡片背景：`#1a1f3a` (深蓝灰)
  - 文字颜色：`#e8eaed` (浅灰白)

- **设计元素**:
  - 渐变标题效果 (`--gradient-primary`)
  - 悬浮光晕效果 (`--shadow-glow`)
  - 毛玻璃效果的徽章
  - 带有径向渐变的英雄区域背景
  - 圆角卡片设计，带悬浮动画

- **布局结构**:
  ```html
  <section class="hero-section">
    <div class="hero-container">
      <div class="hero-badge">标签</div>
      <h1 class="hero-title">标题</h1>
      <p class="hero-subtitle">副标题</p>
    </div>
  </section>
  <main class="main-content">
    <!-- 内容区域 -->
  </main>
  ```

#### 适用场景
- 技术产品介绍
- 开发工具展示
- 科技类文章

## 📐 通用设计模式

### 1. 英雄区域 (Hero Section)
所有风格都包含一个突出的英雄区域：
- 大标题带渐变效果
- 副标题说明
- 统计数据展示（可选）
- 行动按钮（CTA）

### 2. 内容区域结构
- 主容器最大宽度：`1200px`
- 统一的内外边距系统
- 响应式网格布局

### 3. 交互效果
- 卡片悬浮效果 (`transform: translateY(-5px)`)
- 平滑过渡动画 (`transition: all 0.3s ease`)
- 光晕阴影效果

### 4. 代码展示
- 深色背景代码块
- 语法高亮
- 圆角边框
- 可选的标签页切换

## 🎯 选择建议

### 根据内容类型选择风格

1. **技术产品/工具介绍** → 深色科技风格
   - 突出技术感和专业性
   - 适合展示性能参数

2. **AI/创新类内容** → 紫色渐变风格
   - 传达创新和未来感
   - 适合展示前沿技术

3. **教程/文档** → 统一深色主题
   - 保证阅读舒适度
   - 减少视觉疲劳

4. **企业级平台** → 深色科技风格或紫色渐变风格
   - 体现专业性和可靠性
   - 适合展示企业级功能

### 一致性建议

- 同一章节内保持风格统一
- 导航和页脚使用相同样式
- 颜色系统保持一致性
- 交互效果保持统一
