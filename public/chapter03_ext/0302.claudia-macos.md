## 1. 什么是 Claudia？

Claudia 是一个免费开源的强大 GUI 应用程序，专为 Claude Code 设计，旨在提供终极的视觉界面体验。它将 Claude Code 的命令行功能转化为直观的图形界面，让用户能够更高效地进行编程和项目管理。

### 核心价值

- **可视化界面**：将复杂的命令行操作转化为直观的图形界面
- **智能体管理**：轻松创建和管理自定义 AI 智能体
- **会话管理**：可视化管理和跟踪多个 Claude Code 会话
- **后台运行**：支持安全的后台智能体运行
- **项目可视化**：以可视化方式管理项目结构和文件

## 4. 安装指南

### 4.1 官方版本下载

#### 方法1：直接下载安装包
1. 访问 [GitHub Releases](https://github.com/GoJam11/claudia-macos/releases)
2. 下载最新的 `.dmg` 安装包
3. 双击安装包并拖拽到 Applications 文件夹

#### 方法2：使用 Homebrew Cask
```bash
# 添加 Cask 仓库
brew tap gojam11/claudia

# 安装 Claudia
brew install --cask claudia

# 更新 Claudia
brew upgrade --cask claudia
```

### 4.2 手动编译安装

#### 克隆仓库
```bash
# 克隆 Claudia 仓库
git clone https://github.com/GoJam11/claudia-macos.git

# 进入项目目录
cd claudia-macos
```

#### 安装依赖
```bash
# 安装 Node.js 依赖
npm install

# 安装 Python 依赖（如果需要）
pip install -r requirements.txt
```

#### 构建应用
```bash
# 构建 macOS 应用
npm run build:mac

# 或使用 electron-builder
npm run electron:mac
```

### 4.3 配置 Claude Code

Claudia 需要连接到您的 Claude Code 实例：

```bash
# 确保 Claude Code 已安装
claude --version

# 配置 API 密钥
claude auth login
```

## 5. 使用方法

### 5.1 首次启动配置

1. **启动应用**
   ```bash
   # 从 Applications 文件夹启动
   open /Applications/Claudia.app
   ```

2. **配置 Claude Code 连接**
   - 打开设置面板
   - 输入 Claude Code API 密钥
   - 测试连接状态

3. **选择工作目录**
   - 点击"选择项目文件夹"
   - 选择您的项目根目录
   - 确认项目结构加载

### 5.2 创建自定义智能体

1. **打开智能体创建器**
   - 点击侧边栏"智能体"图标
   - 点击"创建新智能体"

2. **配置智能体参数**
   ```json
   {
     "name": "代码审查助手",
     "description": "专门用于代码审查和优化的智能体",
     "model": "claude-3-opus-20240229",
     "temperature": 0.1,
     "max_tokens": 4000,
     "system_prompt": "你是一个专业的代码审查专家..."
   }
   ```

3. **保存并测试智能体**
   - 点击"保存智能体"
   - 在测试区域验证功能

### 5.3 管理会话

1. **创建新会话**
   - 点击"新建会话"按钮
   - 选择智能体或使用默认配置
   - 开始对话

2. **会话操作**
   - **重命名**：双击会话标签进行重命名
   - **导出**：右键选择"导出会话"
   - **删除**：右键选择"删除会话"
   - **搜索**：使用搜索框查找历史会话

### 5.4 项目文件操作

1. **文件浏览**
   - 在左侧文件树中浏览项目
   - 点击文件查看内容
   - 使用搜索功能快速定位文件

2. **文件编辑**
   - 双击文件打开编辑器
   - 使用语法高亮和代码补全
   - 保存更改自动同步

3. **批量操作**
   - 选择多个文件进行批量操作
   - 右键菜单提供多种操作选项

## 8. 故障排除

### 常见问题

#### 1. 应用无法启动
```bash
# 检查系统权限
sudo spctl --master-disable

# 重新安装应用
rm -rf /Applications/Claudia.app
# 重新下载并安装
```

#### 2. Claude Code 连接失败
```bash
# 检查 Claude Code 版本
claude --version

# 重新认证
claude auth logout
claude auth login

# 检查网络连接
ping api.anthropic.com
```

#### 3. 智能体创建失败
- 检查 API 密钥是否有效
- 确认网络连接正常
- 验证智能体配置格式
- 查看错误日志获取详细信息

#### 4. 文件加载缓慢
```bash
# 清理缓存
rm -rf ~/Library/Application\ Support/Claudia/Cache

# 重启应用
killall Claudia
open /Applications/Claudia.app
```

### 调试模式

启用调试模式获取详细日志：

1. **打开调试控制台**
   - 按下 `Cmd+Option+I`
   - 切换到 Console 标签

2. **查看日志文件**
   ```bash
   # 查看应用日志
   tail -f ~/Library/Logs/Claudia/app.log

   # 查看错误日志
   tail -f ~/Library/Logs/Claudia/error.log
   ```

3. **启用详细日志**
   - 在设置中启用"调试模式"
   - 重启应用以应用设置

### 性能优化建议

1. **定期清理缓存**
   - 每周清理一次应用缓存
   - 删除不使用的项目配置

2. **优化系统资源**
   - 关闭不必要的后台应用
   - 增加虚拟内存（如需要）

3. **网络优化**
   - 使用稳定的网络连接
   - 配置代理（如需要）
