## 1. 项目概述

TrendRadar 是一个强大的全网热点聚合监控工具，专为内容创作者、营销人员和企业决策者设计。通过智能算法和AI分析，TrendRadar能够实时追踪全网热点，提供精准的趋势分析和个性化推送服务。

### 🎯 设计目标

- **快速部署** - 30秒内完成部署和配置
- **智能监控** - AI驱动的内容筛选和趋势分析
- **多源聚合** - 整合多个平台的热点数据
- **个性化推送** - 根据用户兴趣定制推送策略

## 3. 推送模式

### 📊 三大推送模式

#### 当日汇总模式
- **推送时间**: 每日固定时间推送
- **内容格式**: 当日热点汇总报告
- **适用场景**: 日常内容创作参考
- **特点**: 系统性强，便于回顾

```json
// 当日汇总推送示例
{
  "date": "2025-12-22",
  "top_topics": [
    {
      "title": "AI技术新突破",
      "trend_score": 95,
      "platforms": ["微博", "知乎", "B站"],
      "summary": "AI领域出现重大技术突破..."
    }
  ],
  "analysis": "今日热点主要集中在科技领域..."
}
```

#### 当前榜单模式
- **推送时机**: 实时热点达到阈值时推送
- **内容格式**: 热点排行榜单
- **适用场景**: 及时跟进热点
- **特点**: 实时性强，抓住热点窗口期

```json
// 当前榜单推送示例
{
  "timestamp": "2025-12-22T14:30:00Z",
  "real_time_ranking": [
    {
      "rank": 1,

      "heat": "爆",
      "growth_rate": "+500%"
    }
  ],
  "alerts": ["检测到突发热点事件"]
}
```

#### 增量监控模式
- **推送机制**: 监控特定话题的变化
- **内容格式**: 话题变化趋势报告
- **适用场景**: 特定领域深度监控
- **特点**: 精准监控，适合垂直领域

```json

  "topic": "新能源汽车",

    {
```

### 📊 趋势分析算法

#### 热度计算
```typescript
// 热度评分算法
interface TrendMetrics {
  mention_count: number;      // 提及次数
  engagement_rate: number;    // 互动率
  growth_rate: number;        // 增长率
  sentiment_score: number;    // 情感分数
  platform_weight: number;   // 平台权重
}

function calculateTrendScore(metrics: TrendMetrics): number {
  const weights = {
    mention: 0.3,
    engagement: 0.25,
    growth: 0.25,
    sentiment: 0.1,
    platform: 0.1
  };

  return (
    metrics.mention_count * weights.mention +
    metrics.engagement_rate * weights.engagement +
    metrics.growth_rate * weights.growth +
    metrics.sentiment_score * weights.sentiment +
    metrics.platform_weight * weights.platform
  );
}
```

```typescript
async function pushViaWebhook(content: TrendContent, config: WebhookConfig) {
```

#### 实时监控
```typescript
// 实时监控流程
class RealtimeMonitor {
  private websocket: WebSocket;
  private dataSources: DataSource[];
  private analyzers: TrendAnalyzer[];

  async startMonitoring() {
    // 连接数据源
    for (const source of this.dataSources) {
      await source.connect();
      source.on('data', this.processNewData.bind(this));
    }

    // 启动分析器
    this.analyzers.forEach(analyzer => analyzer.start());

    console.log('实时监控已启动');
  }

  private processNewData(data: RawData) {
    // 数据预处理
    const cleanedData = this.cleanData(data);

    // 分析热度
    const analysis = this.analyzeTrend(cleanedData);

    // 检查阈值
    if (this.shouldAlert(analysis)) {
      await this.sendAlert(analysis);
    }

    // 更新数据库
    await this.updateDatabase(analysis);
  }
}
```
