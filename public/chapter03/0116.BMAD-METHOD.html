安装指南

多种部署方式，快速搭建多智能体协作环境

🐳
Docker Compose 部署
部署命令 
# 克隆仓库
git clone https://github.com/bmadcode/BMAD-METHOD.git
cd BMAD-METHOD

# 启动服务
docker-compose up -d

# 验证安装
curl http://localhost:8080/health
☸️
Kubernetes 部署
部署命令 
# 应用 Kubernetes 配置
kubectl apply -f k8s/

# 查看部署状态
kubectl get pods -n bmad-system

# 获取服务地址
kubectl get svc -n bmad-system
🔧
源码安装
安装命令 
# 克隆仓库
git clone https://github.com/bmadcode/BMAD-METHOD.git
cd BMAD-METHOD

# 安装依赖
npm install

# 构建项目
npm run build

# 启动服务
npm start
基础配置示例 (~/.bmad/config.yaml) 
server:
  host: "0.0.0.0"
  port: 8080
  workers: 4

agents:
  registry:
    enabled: true
    auto_discovery: true
    health_check_interval: 30

  communication:
    protocol: "grpc"
    timeout: 30000

workflows:
  default_timeout: 3600
  max_concurrent: 10
  retry_policy:
    max_retries: 3
    backoff: "exponential"

monitoring:
  enabled: true
  metrics_interval: 60
  log_level: "info"
