## 1. 什么是 Cometix-Tab？

Cometix-Tab 是由 Haleclipse 开发的开源项目，属于 Copilot++ 生态系统的一部分。它为流行的代码编辑器（VSCode、JetBrains IDE系列）提供了类似 Cursor Tab 的增强标签页管理功能，让开发者能够享受无限标签页、智能标签页管理等先进功能。

### 核心价值

- **无限标签页**：突破传统IDE的标签页数量限制
- **智能管理**：基于使用模式的智能标签页管理
- **跨平台支持**：支持VSCode和JetBrains IDE系列
- **无需账号**：不需要魔法网络或付费账号
- **开源免费**：完全开源，免费使用

### 设计理念

Cometix-Tab 遵循以用户体验为中心的设计原则：
- **无缝集成**：与原生IDE功能无缝集成
- **性能优先**：高效的内存管理和标签页切换
- **用户友好**：直观的界面和简单的操作
- **可扩展性**：支持插件和自定义扩展

## 3. 系统架构

### 技术架构
```
Cometix-Tab Extension
    ├── Tab Manager Core
    ├── Session Manager
    ├── Search Engine
    ├── UI Components
    └── Configuration Manager
    ↓
IDE Integration Layer
    ├── VSCode Extension API
    ├── JetBrains Platform SDK
    └── Generic IDE Interface
    ↓
Native IDE
    ├── Tab System
    ├── File System
    ├── Project Explorer
    └── Editor Framework
```

### 核心组件

#### Tab Manager Core（标签页管理核心）
- **标签页注册**：管理和注册所有标签页
- **生命周期管理**：管理标签页的创建、激活、关闭
- **状态同步**：与IDE原生标签页状态同步
- **事件处理**：处理标签页相关事件

#### Session Manager（会话管理器）
- **会话序列化**：将标签页状态序列化为可存储格式
- **会话恢复**：从存储中恢复标签页会话
- **自动保存**：自动保存会话状态
- **版本管理**：管理会话版本和变更历史

#### Search Engine（搜索引擎）
- **索引构建**：构建标签页内容索引
- **模糊搜索**：实现高效的模糊搜索算法
- **结果排序**：对搜索结果进行相关性排序
- **高亮显示**：在搜索结果中高亮匹配内容

#### UI Components（UI组件）
- **标签页视图**：自定义标签页显示组件
- **搜索界面**：搜索功能的用户界面
- **配置界面**：插件配置的用户界面
- **快捷键绑定**：快捷键的绑定和处理

## 4. 安装指南

### 4.1 系统要求

#### 支持的IDE
- **Visual Studio Code**：1.80.0 或更高版本
- **JetBrains IDEs**：
  - IntelliJ IDEA：2023.2 或更高版本
  - PyCharm：2023.2 或更高版本
  - WebStorm：2023.2 或更高版本
  - 其他JetBrains IDEs：支持Platform SDK的版本

#### 系统要求
- **操作系统**：Windows 10+, macOS 10.15+, Linux (Ubuntu 18.04+)
- **内存**：4GB RAM（推荐8GB）
- **存储空间**：100MB 可用空间
- **处理器**：64位处理器

### 4.2 VSCode 安装

#### 方法1：从扩展市场安装
```bash
# 在VSCode中
1. 打开扩展面板 (Ctrl+Shift+X)
2. 搜索 "Cometix-Tab"
3. 点击安装
4. 重新加载VSCode
```

#### 方法2：从VSIX文件安装
```bash
# 下载VSIX文件
wget https://github.com/Haleclipse/Cometix-Tab/releases/latest/download/cometix-tab.vsix

# 安装扩展
code --install-extension cometix-tab.vsix

# 重新加载VSCode
```

#### 方法3：源码安装
```bash
# 克隆仓库
git clone https://github.com/Haleclipse/Cometix-Tab.git
cd Cometix-Tab

# 安装依赖
npm install

# 构建扩展
npm run build

# 打包扩展
npm run package

# 安装打包后的扩展
code --install-extension cometix-tab-*.vsix
```

### 4.3 JetBrains IDE 安装

#### 方法1：从插件市场安装
```bash
# 在JetBrains IDE中
1. 打开 File -> Settings -> Plugins
2. 搜索 "Cometix-Tab"
3. 点击安装
4. 重启IDE
```

#### 方法2：从本地文件安装
```bash
# 下载插件文件
wget https://github.com/Haleclipse/Cometix-Tab/releases/latest/download/cometix-tab.zip

# 在IDE中安装
1. 打开 File -> Settings -> Plugins
2. 点击齿轮图标 -> Install Plugin from Disk
3. 选择下载的zip文件
4. 重启IDE
```

### 4.4 验证安装

#### VSCode 验证
```bash
# 检查扩展是否安装
code --list-extensions | grep cometix

# 查看扩展状态
打开VSCode -> 扩展面板 -> 查看Cometix-Tab状态
```

#### JetBrains IDE 验证
```bash
# 查看已安装插件
打开IDE -> File -> Settings -> Plugins -> Installed
# 查找Cometix-Tab
```

## 6. 使用方法

### 6.1 基础使用

#### 启用Cometix-Tab
```bash
# VSCode中
1. 打开命令面板 (Ctrl+Shift+P)
2. 输入 "Cometix-Tab: Enable"
3. 选择启用

# JetBrains IDE中
1. 打开 File -> Settings -> Tools -> Cometix-Tab
2. 勾选 "Enable Cometix-Tab"
3. 点击 Apply
```

#### 查看标签页面板
```bash
# 使用快捷键
Ctrl+Shift+T (默认快捷键)

# 或通过菜单
VSCode: View -> Cometix-Tab Panel
JetBrains: View -> Tool Windows -> Cometix-Tab
```

#### 基础操作
```bash
# 打开新标签页
Ctrl+Shift+O

# 关闭标签页
Ctrl+Shift+W

# 在标签页间切换
Ctrl+Shift+Tab

# 搜索标签页
Ctrl+Shift+F
```

### 6.2 高级功能

#### 项目分组
```javascript
// 自动按项目分组标签页

// 配置项目分组规则
{

    "autoDetect": true,

}
```

## 7. 故障排除

### 常见问题

#### 1. 扩展无法加载
```bash
# 检查VSCode版本
code --version

# 重新安装扩展
code --uninstall-extension haleclipse.cometix-tab
code --install-extension haleclipse.cometix-tab

# 检查扩展日志
打开VSCode -> Help -> Toggle Developer Tools
查看 Console 中的错误信息
```

#### 2. 标签页显示异常
```bash
# 重置配置
在VSCode中打开设置
搜索 "cometixTab"
点击 "Reset to Default"

# 清理缓存
rm -rf ~/.vscode/extensions/haleclipse.cometix-tab-*/
rm -rf ~/.vscode/cometix-tab/
```

#### 3. 搜索功能不工作
```bash
# 重建索引
Cometix-Tab: Rebuild Index

# 检查搜索配置
验证搜索配置是否正确设置

# 清理搜索缓存
Cometix-Tab: Clear Search Cache
```

#### 4. 性能问题
```bash
# 调整性能配置
{
  "cometixTab.performance": {
    "lazyLoading": true,
    "cacheSize": 50,
    "maxMemoryUsage": 256
  }
}

# 监控性能
Cometix-Tab: Show Performance Stats
```

### 调试模式

#### 启用调试日志
```json
{

    "logLevel": "debug",
```

#### 查看调试信息
