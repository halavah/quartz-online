# 01. 项目总览与快速开始

## 📖 项目简介

**Halavah's Tech** 是一个基于 Next.js 15 构建的现代化技术分享平台，专注于 AI 工具、开发技术和前沿科技的分享。支持外部 HTML 文章挂载，配置简单，易于维护。

### 核心亮点

- 🔍 **炫酷搜索功能** - 全屏弹窗式搜索，毛玻璃背景，实时搜索结果展示
- 🌓 **主题切换** - 支持暗色/明亮主题，自动保存用户偏好
- 🏷️ **动态分类筛选** - 智能分类系统，自动从文章提取分类，快速筛选
- 📊 **网格/列表视图** - 灵活的视图切换，满足不同浏览习惯
- 📄 **分页导航** - 优雅的分页体验，每页 6 篇文章
- 📱 **响应式设计** - 完美适配桌面和移动端
- 🚀 **静态站点生成** - 构建后生成纯静态文件，部署简单
- ⚙️ **集中配置管理** - 所有文章和站点配置通过 JSON 文件统一管理
- 🎯 **可关闭广告横幅** - 导航栏广告位，支持关闭，不影响用户体验

## ✨ 核心特性详解

### 1. 炫酷搜索功能

点击导航栏搜索图标，弹出全屏搜索界面：
- **毛玻璃背景** - 半透明黑色背景 + 8px 模糊效果
- **渐变搜索框** - 根据主题自动切换渐变背景
- **实时搜索** - 支持搜索标题、描述、分类，最多显示 8 条结果
- **精美卡片** - 搜索结果以卡片形式展示，hover 高亮
- **快捷键支持** - ESC 关闭，输入框自动聚焦

### 2. 主题切换系统

- 暗色主题（默认）和明亮主题
- 一键切换，即时生效
- localStorage 持久化保存
- 所有组件自动适配主题

### 3. 动态分类系统

- 自动从 articles.json 提取文章分类
- 支持多级分类
- 分类按钮显示文章数量
- 点击分类即可筛选相关文章

### 4. 视图模式切换

- **网格视图** - 卡片式布局，信息展示丰富
- **列表视图** - 简洁布局，适合快速浏览
- 视图状态自动保存

### 5. 集中配置管理

所有站点配置和文章元数据集中在 `data/articles.json`：
- `siteName` - 站点名称
- `siteDescription` - 站点描述
- `beianNumber` - 备案号
- `copyrightText` - 版权信息
- `adLink` - 广告链接
- `articles` - 文章列表

## 🚀 快速开始

### 前置要求

- Node.js 18.0 或更高版本
- npm 或 yarn 包管理器
- 基础的 HTML/TypeScript 知识

### 安装步骤

#### 1. 克隆项目

```bash
git clone https://github.com/halavah/quartz-online.git
cd quartz-online
```

#### 2. 安装依赖

```bash
npm install
```

#### 3. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000 查看您的博客！

#### 4. 构建生产版本

```bash
npm run build
```

构建完成后，静态文件将输出到 `out/` 目录。

### 常用命令

```bash
# 开发模式（热重载）
npm run dev

# 构建生产版本（静态导出）
npm run build

# 启动生产服务器（需要先 build）
npm start

# TypeScript 类型检查
npm run type-check

# 清理缓存
rm -rf .next node_modules/.cache
```

## 📝 快速发布第一篇文章

### 步骤 1: 创建 HTML 文章

在 `public/` 目录下创建您的文章文件，例如：

```bash
public/
└── chapter01/
    └── 01.my-first-article.html
```

HTML 文章示例：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>我的第一篇文章</title>
    <style>
        body {
            font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
            line-height: 1.6;
            max-width: 800px;
            margin: 0 auto;
            padding: 40px;
        }
        h1 { color: #1a202c; font-size: 2.5rem; margin-bottom: 1rem; }
        h2 { color: #2d3748; font-size: 2rem; margin-top: 2rem; }
        p { color: #4a5568; font-size: 1.1rem; margin-bottom: 1rem; }
    </style>
</head>
<body>
    <h1>深入理解 JavaScript 异步编程</h1>

    <h2>什么是异步编程？</h2>
    <p>异步编程是一种编程范式，允许程序在等待某些操作完成时继续执行其他任务...</p>

    <h2>Promise 的使用</h2>
    <p>Promise 是 JavaScript 中处理异步操作的强大工具...</p>
</body>
</html>
```

### 步骤 2: 更新配置文件

编辑 `data/articles.json`，添加您的文章信息：

```json
{
  "config": {

    "siteDescription": "探索前沿技术，提升开发效率",
    "githubUrl": "https://github.com/halavah",
    "beianNumber": "京ICP备12345678号",

    "adLink": "https://api.nekoapi.com"
  },
  "articles": [
    {
      "title": "深入理解 JavaScript 异步编程",
```

```bash
npm run build
```

## 🎨 自定义博客

### 修改站点信息

编辑 `data/articles.json` 中的 `config` 部分：

```json
{
  "config": {
    "siteName": "我的技术博客",
    "siteDescription": "分享技术心得，记录成长历程",
    "githubUrl": "https://github.com/your-username"
  }
}
```

### 修改主题颜色

编辑 `app/globals.css`，自定义 CSS 变量：

```css
:root[data-theme="dark"] {
  --dark-bg: #0a0e27;              /* 深色背景 */
  --card-bg: #1a1f3a;              /* 卡片背景 */
  --primary-color: #0066ff;         /* 主色调 */
  --gradient-primary: linear-gradient(135deg, #0066ff 0%, #00d4ff 100%);
}

:root[data-theme="light"] {
  --dark-bg: #ffffff;              /* 明亮背景 */
  --card-bg: #f8f9fa;              /* 卡片背景 */
  --primary-color: #0066ff;         /* 主色调 */
}
```

### 调整分页数量

在 `app/page.tsx` 中修改常量：

```typescript
const ARTICLES_PER_PAGE = 6; // 修改为您想要的数字（建议 6-12）
```
