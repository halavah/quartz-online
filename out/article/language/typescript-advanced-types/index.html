TypeScript 高级类型技巧

TypeScript 的类型系统非常强大，掌握高级类型技巧可以让你的代码更加健壮和可维护。本文将介绍几个实用的高级类型技巧。

1. 条件类型 (Conditional Types)

条件类型允许我们根据类型关系来选择类型：

// 基础条件类型
type IsString<T> = T extends string ? true : false;

// 使用示例
type Test1 = IsString<string>  // true
type Test2 = IsString<number>  // false

// 实际应用：提取数组元素类型
type ArrayElement<T> = T extends (infer U)[] ? U : never;

type Numbers = ArrayElement<number[]>  // number

2. 映射类型 (Mapped Types)

映射类型可以创建新的类型，基于现有类型的结构：

// 将所有属性变为可选
type Partial<T> = {
  [P in keyof T]?: T[P];
};

// 将所有属性变为只读
type Readonly<T> = {
  readonly [P in keyof T]: T[P];
};

// 实际应用：创建更新类型
type Update<T> = Partial<Pick<T, Exclude<keyof T, 'id'>>>;

interface User {
  id: number;
  name: string;
  email: string;
}

type UserUpdate = Update<User>;
// 等价于: { name?: string; email?: string; }

3. 模板字面量类型 (Template Literal Types)

TypeScript 4.1+ 支持模板字面量类型：

type EventName = `on${Capitalize<string>}`;

// 创建事件处理器类型
type EventHandlers<T> = {
  [K in keyof T as K extends string
    ? `on${Capitalize<K>}`
    : never]: T[K] extends (...args: any[]) => any
      ? T[K]
      : (event: any) => void;
};

// 使用示例
interface Events {
  click: (e: MouseEvent) => void;
  scroll: (e: Event) => void;
}

type Handlers = EventHandlers<Events>;
// 等价于: { onClick: (e: MouseEvent) => void; onScroll: (e: Event) => void; }

4. 递归类型

示例：深度只读 
type DeepReadonly<T> = {
  readonly [P in keyof T]: T[P] extends object
    ? DeepReadonly<T[P]>
    : T[P];
};

interface Nested {
  user: {
    name: string;
    address: {
      street: string;
    };
  };
}

type DeepNested = DeepReadonly<Nested>;
// 所有嵌套属性都变为只读

5. 工具类型组合

技巧：组合多个工具类型创建复杂类型 
// 提取函数返回类型
type ReturnType<T> = T extends (...args: any[]) => infer R ? R : never;

// 提取 Promise 的值类型
type UnpackPromise<T> = T extends Promise<infer U> ? U : T;

// 组合使用：异步函数的返回类型
type AsyncReturnType<T> = UnpackPromise<ReturnType<T>>;

async function fetchUser(): Promise<{ id: number; name: string }> {
  return { id: 1, name: 'John' };
}

type UserType = AsyncReturnType<typeof fetchUser>;
// { id: number; name: string }

6. 类型守卫和类型谓词

// 类型谓词函数
function isString(value: unknown): value is string {
  return typeof value === 'string';
}

// 使用
function processValue(value: unknown) {
  if (isString(value)) {
    // TypeScript 知道这里 value 是 string
    console.log(value.toUpperCase());
  }
}

// 更复杂的类型守卫
function isUser(obj: any): obj is User {
  return obj && typeof obj.id === 'number' && typeof obj.name === 'string';
}

7. 使用 infer 提取类型

// 从函数参数提取类型
type FirstArg<T> = T extends (first: infer F, ...args: any[]) => any ? F : never;

type Test1 = FirstArg<(name: string, age: number) => void>;  // string

// 从 Promise 提取类型
type Awaited<T> = T extends Promise<infer U> ? U : T;

type Test2 = Awaited<Promise<string[]>>;  // string[]

8. Brand Types

用途：创建不能与其他类型互换的唯一类型 
interface Brand<T> {
  __type: T;
  __brand: unique symbol;
}

type BrandId = Brand<'ID'>;

function createId(id: number): BrandId {
  return id as any;
}

// 防止误用
function processId(id: BrandId) {
  console.log(id);
}

// processId(123); // 错误！
processId(createId(123)); // 正确

总结

TypeScript 的高级类型系统提供了强大的类型操作能力。通过合理使用这些技巧，你可以：

  • 编写类型安全的代码
  • 提供更好的 IDE 支持
  • 减少运行时错误
  • 提高代码的可维护性
