快速开始

多种部署方式，2分钟即可启动

方式1: Docker Compose (推荐)

完整部署 
# 克隆项目
git clone https://github.com/bytebot-ai/bytebot.git
cd bytebot

# 配置AI提供商（选择一个）
echo "ANTHROPIC_API_KEY=sk-ant-..." > docker/.env

# 启动所有服务（桌面、代理、UI、数据库）
docker-compose -f docker/docker-compose.yml up -d

# 访问服务
# - 任务界面: http://localhost:9992
# - 代理API: http://localhost:9991
# - 桌面API: http://localhost:9990
# - VNC访问: http://localhost:9990/vnc

方式2: Railway 一键部署

  • 访问 Railway 部署页面
  • 点击 "Deploy Now" 按钮
  • 添加环境变量（选择一个AI提供商）
  • 点击 Deploy，等待2-3分钟
  • Railway会自动构建所有服务并生成公共URL

API集成指南

三种API方式，满足不同集成需求

1. Agent API - 高级任务接口

创建AI任务 (Python)

Python示例 
import requests

# 创建任务
response = requests.post(
    "http://localhost:9991/tasks",
    json={
        "instruction": "登录我的银行账户，下载上月的对账单",
        "files": []
    }
)

task = response.json()
task_id = task["id"]

# 检查任务状态
import time
while True:
    status = requests.get(f"http://localhost:9991/tasks/{task_id}")
    task_data = status.json()

    if task_data['status'] in ['completed', 'failed']:
        print(f"结果: {task_data['result']}")
        break

    time.sleep(2)

2. Computer Use API - 直接桌面控制

截图和鼠标操作

Python示例 
import requests

BASE_URL = "http://localhost:9990"

# 获取屏幕截图
screenshot = requests.post(f"{BASE_URL}/computer-use", json={
    "action": "screenshot"
})

# 移动鼠标并点击
requests.post(f"{BASE_URL}/computer-use", json={
    "action": "mouse_move",
    "coordinate": [100, 200]
})

requests.post(f"{BASE_URL}/computer-use", json={
    "action": "left_click"
})

3. MCP协议 - Claude Desktop集成

通过Model Context Protocol连接Claude Desktop，让Claude直接控制桌面

Claude Desktop配置 
# macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
# Windows: %APPDATA%\Claude\claude_desktop_config.json

{
  "mcpServers": {
    "bytebot": {
      "url": "http://localhost:9990/mcp"
    }
  }
}
