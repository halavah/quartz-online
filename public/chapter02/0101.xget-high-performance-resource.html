快速开始

无需注册，三步即可开始使用 Xget 加速服务

GitHub 资源加速

示例：加速 GitHub 文件下载

原始 URL 
https://github.com/microsoft/vscode/archive/refs/heads/main.zip
加速 URL 
https://xget.xi-xu.me/gh/microsoft/vscode/archive/refs/heads/main.zip

Git 仓库克隆加速

直接使用 git clone 命令

加速克隆 
git clone https://xget.xi-xu.me/gh/microsoft/vscode.git
提示：将 github.com 替换为 xget.xi-xu.me/gh 即可实现加速

包管理器配置

一次配置，永久加速。支持 npm、pip、Docker 等主流包管理器

npm 镜像配置

配置命令 
# 设置 npm 镜像源
npm config set registry https://xget.xi-xu.me/npm/

# 验证配置
npm config get registry

pip 镜像配置

配置命令 
# 设置 pip 镜像源
pip config set global.index-url https://xget.xi-xu.me/pypi/simple/
pip config set global.trusted-host xget.xi-xu.me

Git 全局配置

自动加速所有 Git 操作 
# GitHub 加速
git config --global url."https://xget.xi-xu.me/gh/".insteadOf "https://github.com/"

# GitLab 加速
git config --global url."https://xget.xi-xu.me/gl/".insteadOf "https://gitlab.com/"
自动加速：配置后，所有 git clone、git pull 等操作都将自动使用加速服务

Docker 镜像加速

使用示例 
# 拉取 GitHub Container Registry 镜像
docker pull xget.xi-xu.me/cr/ghcr/nginxinc/nginx-unprivileged:latest

AI API 加速

加速 OpenAI、Anthropic、Gemini 等 AI 服务的 API 访问

OpenAI API 加速

URL 替换 
# 原始 URL
https://api.openai.com/v1/chat/completions

# 加速 URL
https://xget.xi-xu.me/ip/openai/v1/chat/completions
Python 使用示例 
import openai

# 配置加速端点
openai.api_base = "https://xget.xi-xu.me/ip/openai"
openai.api_key = "your-api-key"

# 正常调用
response = openai.ChatCompletion.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "Hello!"}]
)

自部署到 Cloudflare Workers

完全免费，每天 10 万次请求额度，拥有独立加速域名

部署步骤

命令行部署 
# 1. 克隆项目
git clone https://github.com/xixu-me/Xget.git
cd Xget

# 2. 安装依赖
npm install

# 3. 登录 Cloudflare
npx wrangler login

# 4. 部署到 Workers
npm run deploy
部署完成！你将获得一个专属加速域名，例如：https://xget.your-subdomain.workers.dev

自部署优势

  • 完全免费 - Cloudflare Workers 免费套餐每天 10 万次请求
  • 全球加速 - 自动部署到全球 330+ 边缘节点
  • 自定义域名 - 可绑定自己的域名
  • 数据安全 - 完全控制，数据不经过第三方
  • 零维护成本 - Cloudflare 自动处理扩展和维护
