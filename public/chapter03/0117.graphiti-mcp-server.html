安装指南

快速部署Neo4j知识图谱服务器

🐳
Docker Compose 部署
部署命令 
# 克隆仓库
git clone https://github.com/gifflet/graphiti-mcp-server.git
cd graphiti-mcp-server

# 启动服务（包含 Neo4j）
docker-compose up -d

# 验证安装
curl http://localhost:8080/health
🔧
独立部署
安装命令 
# 安装 Neo4j
# 参考官方文档：https://neo4j.com/docs/

# 克隆并构建 Graphiti
git clone https://github.com/gifflet/graphiti-mcp-server.git
cd graphiti-mcp-server

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env

# 启动服务
npm start
⚙️
Neo4j 配置
环境变量配置 
# .env 文件
NEO4J_URI=bolt://localhost:7687
NEO4J_USER=neo4j
NEO4J_PASSWORD=password
NEO4J_DATABASE=neo4j

GRAPHITI_SERVER_PORT=8080
GRAPHITI_LOG_LEVEL=info

REDIS_URL=redis://localhost:6379
CACHE_TTL=3600

MCP_SERVER_NAME=graphiti-mcp
MCP_SERVER_VERSION=1.0.0
MCP 配置示例 (~/.claude/settings.json) 
{
  "mcpServers": {
    "graphiti": {
      "command": "node",
      "args": ["/path/to/graphiti-mcp-server/dist/index.js"],
      "env": {
        "NEO4J_URI": "bolt://localhost:7687",
        "NEO4J_USER": "neo4j",
        "NEO4J_PASSWORD": "password"
      }
    }
  }
}
