# 快速开始指南

本指南将帮助您在 5 分钟内搭建并运行 TechVerse 博客系统。

## 📋 前置要求

- Node.js 18+
- npm 或 yarn 包管理器
- 基础的 HTML 知识

## 🚀 快速部署

### 1. 克隆项目

```bash
git clone <repository-url>
cd media-online
```

### 2. 安装依赖

```bash
npm install
```

### 3. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000 查看您的博客！

### 4. 构建生产版本

```bash
npm run build
```

构建完成后，静态文件将输出到 `out/` 目录。

## 📝 发布第一篇文章

### 步骤 1: 创建 HTML 文章

在 `public/` 目录下创建您的文章文件。建议按分类组织：

```bash
# 例如：创建一篇技术文章
public/
└── tech/
    └── my-first-article.html
```

HTML 文件示例：

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>我的第一篇技术文章</title>
    <style>
        body { font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif; }
        .container { max-width: 800px; margin: 0 auto; padding: 40px; }
        h1 { color: #1a202c; }
        .meta { color: #718096; font-size: 14px; margin-bottom: 20px; }
    </style>
</head>
<body>
    <div class="container">
        <h1>深入理解 JavaScript 异步编程</h1>
        <div class="meta">发布于 2024-12-12 | 阅读时间 10 分钟</div>

        <h2>什么是异步编程？</h2>
        <p>异步编程是一种编程范式，允许程序在等待某些操作完成时继续执行其他任务...</p>

        <!-- 更多内容 -->
    </div>
</body>
</html>
```

### 步骤 2: 更新配置文件

编辑 `data/articles.json`，添加您的文章信息：

```json
{
  "config": {
    "siteName": "TechVerse",
    "siteDescription": "探索技术的无限可能"
  },
  "articles": [
    {
      "title": "深入理解 JavaScript 异步编程",
      "description": "本文将详细介绍 JavaScript 中的异步编程概念、Promise 和 async/await 的使用。",
      "htmlFile": "tech/my-first-article.html"
    }
  ]
}
```

### 步骤 3: 重新构建

```bash
npm run build
```

您的文章现在已经发布成功！

## 🎨 自定义博客

### 修改网站信息

编辑 `data/articles.json` 中的 `config` 部分：

```json
{
  "config": {
    "siteName": "我的技术博客",
    "siteDescription": "分享技术心得，记录成长历程"
  }
}
```

### 调整样式

1. **全局样式**: 编辑 `app/globals.css`
2. **Tailwind 配置**: 编辑 `tailwind.config.ts`
3. **页面样式**: 编辑对应的 `.tsx` 文件

### 修改每页文章数量

在 `app/page.tsx` 中修改 `ARTICLES_PER_PAGE` 常量：

```typescript
const ARTICLES_PER_PAGE = 10; // 修改为您想要的数字
```

## 🔧 常用命令

```bash
# 开发模式（热重载）
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start

# 类型检查
npm run type-check

# 代码格式化
npm run format
```

## 📂 文件组织建议

为了让博客结构更清晰，建议按以下方式组织文章：

```
public/
├── tech/                  # 技术文章
│   ├── frontend/         # 前端技术
│   ├── backend/          # 后端技术
│   └── devops/           # DevOps
├── tutorial/             # 教程系列
├── notes/               # 学习笔记
└── thoughts/            # 随想随笔
```
