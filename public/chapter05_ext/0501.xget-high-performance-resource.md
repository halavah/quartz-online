## 1. 项目概述

Xget 是一个超高性能、安全的一站式开发者资源加速引擎，基于 Cloudflare Workers 构建。它利用 Cloudflare 全球 330+ 边缘节点，为开发者提供极速、稳定、安全的资源访问体验。

## 4. 快速开始

### GitHub 资源加速

#### 文件下载示例
```bash
# 原始 URL
https://github.com/microsoft/vscode/archive/refs/heads/main.zip

# 加速 URL
https://xget.xi-xu.me/gh/microsoft/vscode/archive/refs/heads/main.zip
```

### Git 仓库克隆加速

#### 直接使用 git clone 命令
```bash
# 加速克隆
git clone https://xget.xi-xu.me/gh/microsoft/vscode.git
```

> 💡 **提示**：将 `github.com` 替换为 `xget.xi-xu.me/gh` 即可实现加速

## 5. 包管理器配置

### npm 镜像配置
```bash
# 设置 npm 镜像源
npm config set registry https://xget.xi-xu.me/npm/

# 验证配置
npm config get registry
```

### pip 镜像配置
```bash
# 设置 pip 镜像源
pip config set global.index-url https://xget.xi-xu.me/pypi/simple/
pip config set global.trusted-host xget.xi-xu.me
```

### Git 全局配置
```bash
# GitHub 加速
git config --global url."https://xget.xi-xu.me/gh/".insteadOf "https://github.com/"

# GitLab 加速
git config --global url."https://xget.xi-xu.me/gl/".insteadOf "https://gitlab.com/"
```

> ✅ **自动加速**：配置后，所有 `git clone`、`git pull` 等操作都将自动使用加速服务

### Docker 镜像加速
```bash
# 拉取 GitHub Container Registry 镜像
docker pull xget.xi-xu.me/cr/ghcr/nginxinc/nginx-unprivileged:latest
```

## 9. 技术特性

### 🔧 技术实现

- **🌐 HTTP/3 协议** - 基于 QUIC 传输，连接延迟降低 40%，弱网环境表现优异
- **📦 智能压缩** - 支持 gzip、brotli 压缩，自动选择最优算法，效率提升 60%
- **🔗 零延迟预连接** - 连接预热机制，消除 TCP/TLS 握手开销，毫秒级响应
- **⚡ 并行分片下载** - 支持 Range 请求，多线程并行下载，大文件速度倍增
- **🎯 智能路由** - 基于 Argo 智能路由，自动选择最优路径，避开拥堵节点
- **🔄 自动重试** - 最大 3 次重试，智能错误恢复，提高下载成功率
