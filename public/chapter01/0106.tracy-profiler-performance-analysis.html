核心特性

Tracy 提供了全面的性能分析功能，帮助开发者优化应用程序性能

⏱️

纳秒级精度

提供纳秒级的时间分辨率，能够精确测量最微小的性能差异， 适合高性能应用和游戏的精确优化。

🔄

实时分析

实时收集和显示性能数据，无需停止应用程序即可查看性能瓶颈， 支持热重载和动态更新。

🖥️

CPU & GPU 分析

同时支持 CPU 和 GPU 性能分析，包括渲染管线、计算着色器、 内存带宽等多个维度的性能指标。

💾

内存追踪

实时监控内存分配和释放，检测内存泄漏，分析内存使用模式， 支持自定义内存标记和分组。

🔒

锁分析

监控互斥锁和其他同步原语的性能，识别死锁和竞争条件， 优化多线程应用的并发性能。

📊

可视化界面

直观的图形化界面，提供火焰图、时间线、统计图表等多种 数据可视化方式，便于性能分析。

快速开始

只需几行代码即可集成 Tracy

C++ 集成示例
#include "Tracy.hpp"

// 创建性能区域
void GameLoop() {
    ZoneScoped;  // 自动命名的性能区域

    // 命名的性能区域
    ZoneScopedN("Physics Update");

    // 自定义文本区域
    TracyText("FPS", "60");

    // 内存分配追踪
    void* ptr = TracyAlloc(1024, 12);

    // ... 你的代码 ...

    TracyFree(ptr);
}
配置文件示例
// tracy.cpp
#define TRACY_ENABLE
#define TRACY_ON_DEMAND

#include "common/TracySystem.hpp"

tracy::SourceHistogramData g_sourceHistogram;
int main(int argc, char** argv) {
    tracy::InitRuntime();

    // 初始化 Tracy
    ___tracy_set_thread_name("Main Thread");

    // 启动应用程序
    RunApplication();

    return 0;
}
