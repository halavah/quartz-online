项目概述

Tracy 是一个超高性能、安全的实时性能分析器，专为游戏和其他复杂应用程序设计。 它提供纳秒级的时间分辨率测量能力，支持 CPU、GPU、内存、锁等多个维度的性能分析。

性能指标

Tracy 能够测量的关键性能指标

时间测量

  • 纳秒级精度：0.1ns 最小时间精度
  • 高吞吐量：每秒超过 100 万次采样
  • 低开销：CPU 开销小于 0.1%
  • 无限区域：支持无限数量的性能区域

CPU 分析

  • 线程追踪：监控所有线程的执行情况
  • 调用栈：完整的函数调用栈信息
  • CPU 使用率：实时 CPU 使用率统计
  • 上下文切换：线程上下文切换分析

GPU 分析

  • 渲染管线：OpenGL、Vulkan、DirectX 支持
  • GPU 时间：精确的 GPU 执行时间测量
  • 绘制调用：渲染批次和绘制调用统计
  • 着色器分析：顶点、片段、计算着色器性能

内存分析

  • 实时监控：内存分配和释放实时追踪
  • 泄漏检测：自动检测内存泄漏
  • 使用模式：内存使用热图和趋势分析
  • 自定义标记：支持自定义内存标记和分组

快速开始

只需几行代码即可集成 Tracy

C++ 集成示例

main.cpp 
#include "Tracy.hpp"

// 创建性能区域
void GameLoop() {
    ZoneScoped;  // 自动命名的性能区域

    // 命名的性能区域
    ZoneScopedN("Physics Update");

    // 自定义文本区域
    TracyText("FPS", "60");

    // 内存分配追踪
    void* ptr = TracyAlloc(1024, 12);

    // ... 你的代码 ...

    TracyFree(ptr);
}

配置文件示例

tracy.cpp 
// tracy.cpp
#define TRACY_ENABLE
#define TRACY_ON_DEMAND

#include "common/TracySystem.hpp"

tracy::SourceHistogramData g_sourceHistogram;
int main(int argc, char** argv) {
    tracy::InitRuntime();

    // 初始化 Tracy
    ___tracy_set_thread_name("Main Thread");

    // 启动应用程序
    RunApplication();

    return 0;
}

编译配置

CMakeLists.txt 
# 添加 Tracy 库
add_subdirectory(tracy)

# 链接 Tracy
target_link_libraries(YourApp PRIVATE TracyClient)

# 启用 Tracy
target_compile_definitions(YourApp PRIVATE TRACY_ENABLE)
