项目概述

Onyx 是一个易部署的开源 AI 平台，支持空气隔离环境。它提供功能丰富的聊天 UI， 支持任意 LLM（大语言模型），内置代理、Web 搜索、RAG（检索增强生成）等高级功能。

Onyx 适用于从个人到全球企业的团队，提供企业级搜索、安全性和管理 UI。 核心价值包括：自定义代理、40+ 知识源连接器、深度研究能力。

核心优势：完全开源、支持私有部署、兼容所有主流 LLM、提供企业级安全和管理功能

安装指南

支持 Docker、Kubernetes、Terraform 等多种部署方式

一键安装脚本

最简单的部署方式，适合快速体验和小型部署

Bash 
# 下载并运行一键安装脚本
curl -fsSL https://raw.githubusercontent.com/onyx-dot-app/onyx/main/deployment/docker_compose/install.sh > install.sh
chmod +x install.sh
./install.sh
安装完成后：访问 http://localhost:3000 即可开始使用 Onyx

手动 Docker Compose 部署

docker-compose.yml 
version: '3.8'
services:
  api:
    image: onyxdotapp/onyx-backend:latest
    ports:
      - "8080:8080"
    environment:
      - POSTGRES_HOST=postgres
      - POSTGRES_DB=onyx
      - POSTGRES_USER=onyx
      - POSTGRES_PASSWORD=your_password
    depends_on:
      - postgres

  web:
    image: onyxdotapp/onyx-web:latest
    ports:
      - "3000:3000"
    environment:
      - API_URL=http://api:8080

  postgres:
    image: postgres:15
    environment:
      - POSTGRES_DB=onyx
      - POSTGRES_USER=onyx
      - POSTGRES_PASSWORD=your_password
    volumes:
      - postgres_data:/var/lib/postgresql/data

volumes:
  postgres_data:

使用 Helm Chart 部署

适合生产环境的 Kubernetes 部署

Bash 
# 添加 Helm 仓库
helm repo add onyx https://charts.onyx.app
helm repo update

# 安装 Onyx
helm install onyx onyx/onyx \
  --set postgresql.enabled=true \
  --set ingress.enabled=true \
  --set ingress.hosts[0].host=onyx.yourdomain.com

详细配置请参考 Kubernetes 部署文档

使用示例

通过 Web UI 或 API 使用 Onyx

通过 OpenAI SDK 调用 Onyx API

Onyx 提供 OpenAI 兼容的 API，可以使用 OpenAI SDK 直接调用

Python 
from openai import OpenAI
import os

# 配置 Onyx API
client = OpenAI(
    api_key=os.environ.get("OPENAI_API_KEY"),
    base_url="http://localhost:8080/openai-assistants",
    default_headers={
        "Authorization": f"Bearer {os.environ.get('ONYX_API_KEY')}"
    },
)

# 创建助手
assistant = client.beta.assistants.create(
    name="Topic Analyzer",
    instructions="You are a helpful assistant that analyzes topics by searching through workplace documents.",
    tools=[{"type": "SearchTool"}],
    model="gpt-4o",
)

# 创建对话线程
thread = client.beta.threads.create()

# 发送消息
message = client.beta.threads.messages.create(
    thread_id=thread.id,
    role="user",
    content="Analyze the recent developments in AI."
)

# 运行助手
run = client.beta.threads.runs.create(
    thread_id=thread.id,
    assistant_id=assistant.id
)

print(f"Assistant created: {assistant.id}")

创建自定义代理

在 Web UI 中创建具有特定知识和功能的代理

  • 访问 Onyx Web UI
  • 点击"创建代理"按钮

  • 添加知识源（文档、连接器等）
  • 配置可用的工具（Web 搜索、代码解释器等）
  • 保存并开始使用
