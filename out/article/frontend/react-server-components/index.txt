    <h1>React Server Components 实践指南</h1>

    <p>React Server Components（RSC）是 React 团队推出的革命性特性，它允许组件在服务器端渲染，带来更好的性能和开发体验。本文将深入探讨 RSC 的核心概念和实际应用。</p>

    <h2>什么是 Server Components？</h2>
    <p>Server Components 是一种特殊的 React 组件，它们在服务器上运行，而不是在客户端。这使得它们能够直接访问后端资源，同时保持组件化的开发体验。</p>

    <h2>核心优势</h2>
    <div class="tip">
        <strong>主要优势：</strong>
        <ul>
            <li>零客户端 JavaScript 包大小</li>
            <li>直接访问服务器资源（数据库、文件系统等）</li>
            <li>自动代码分割</li>
            <li>更好的首屏加载性能</li>
        </ul>
    </div>

    <h2>基本语法</h2>
    <p>Server Components 使用 <code>'use server'</code> 指令来标识：</p>

    <pre><code>// ServerComponent.server.js
'use server'

import { db } from './database'

export default async function PostList() {
  const posts = await db.posts.findMany()

  return (
    &lt;div&gt;
      {posts.map(post => (
        &lt;div key={post.id}&gt;
          &lt;h2&gt;{post.title}&lt;/h2&gt;
          &lt;p&gt;{post.content}&lt;/p&gt;
        &lt;/div&gt;
      ))}
    &lt;/div&gt;
  )
}</code></pre>

    <h2>与 Client Components 的区别</h2>
    <div class="warning">
        <strong>注意：</strong> Server Components 不能使用 hooks、事件处理器或浏览器 API。需要交互的组件应该使用 Client Components。
    </div>

    <pre><code>// ClientComponent.client.js
'use client'

import { useState, useEffect } from 'react'

export default function Counter() {
  const [count, setCount] = useState(0)

  return (
    &lt;button onClick={() => setCount(count + 1)}&gt;
      Count: {count}
    &lt;/button&gt;
  )
}</code></pre>

    <h2>最佳实践</h2>
    <h3>1. 混合使用 Server 和 Client Components</h3>
    <p>在 Server Components 中嵌入 Client Components：</p>
    <pre><code>// PostPage.server.js
import Counter from './Counter.client'

export default function PostPage({ post }) {
  return (
    &lt;article&gt;
      &lt;h1&gt;{post.title}&lt;/h1&gt;
      &lt;p&gt;{post.content}&lt;/p&gt;
      &lt;Counter /&gt;
    &lt;/article&gt;
  )
}</code></pre>

    <h3>2. 数据获取模式</h3>
    <p>利用 Server Components 的优势进行数据获取：</p>
    <pre><code>// UserProfile.server.js
async function getUser(id) {
  const res = await fetch(`https://api.example.com/users/${id}`)
  return res.json()
}

export default async function UserProfile({ userId }) {
  const user = await getUser(userId)

  return (
    &lt;div&gt;
      &lt;h1&gt;{user.name}&lt;/h1&gt;
      &lt;p&gt;{user.email}&lt;/p&gt;
    &lt;/div&gt;
  )
}</code></pre>

    <h2>性能优化技巧</h2>
    <ul>
        <li>使用 Suspense 处理加载状态</li>
        <li>合理拆分组件，避免不必要的重渲染</li>
        <li>利用 Streaming SSR 提升感知性能</li>
    </ul>

    <h2>总结</h2>
    <p>React Server Components 代表了 React 的未来方向。通过将组件分层（Server vs Client），我们能够构建更高效、更易于维护的应用。在实际项目中，合理使用 Server Components 可以显著提升应用性能。</p>
