安装指南

多种安装方式，快速部署使用

📦
预编译二进制文件
安装命令 
# Linux/macOS 下载
wget https://github.com/lich0821/ccNexus/releases/latest/download/ccnexus-linux-amd64.tar.gz
tar -xzf ccnexus-linux-amd64.tar.gz

# 移动到系统路径
sudo mv ccnexus /usr/local/bin/

# 验证安装
ccnexus --version
🔧
源码编译
编译命令 
# 克隆仓库
git clone https://github.com/lich0821/ccNexus.git
cd ccNexus

# 编译
go build -o ccnexus cmd/ccnexus/main.go

# 安装到系统路径
sudo mv ccnexus /usr/local/bin/

# 验证安装
ccnexus --version
🐳
Docker部署
Docker命令 
# 拉取镜像
docker pull lich0821/ccnexus

# 运行容器
docker run -d \
  --name ccnexus \
  -p 8080:8080 \
  -v $(pwd)/config:/app/config \
  lich0821/ccnexus

# 验证部署
curl http://localhost:8080/health
基础配置示例 (config.yaml) 
server:
  host: "0.0.0.0"
  port: 8080
  workers: 4

providers:
  openai:
    name: "OpenAI"
    base_url: "https://api.openai.com/v1"
    api_key: "${OPENAI_API_KEY}"
    models: ["gpt-4", "gpt-3.5-turbo"]
    timeout: 30000
    weight: 2

  anthropic:
    name: "Anthropic"
    base_url: "https://api.anthropic.com"
    api_key: "${ANTHROPIC_API_KEY}"
    models: ["claude-3-opus-20240229", "claude-3-sonnet-20240229"]
    timeout: 30000
    weight: 3

routing:
  default_strategy: "weighted_round_robin"
  health_check_interval: 30
  max_retries: 3
  timeout: 60000

monitoring:
  enabled: true
  metrics_interval: 60
  log_level: "info"
  export_prometheus: true
