核心功能特性

Claude Code Viewer 提供完整的可视化开发体验

📊

会话日志可视化

将命令行日志转换为直观的Web界面，支持完整对话历史展示和时间线视图

🗂️

会话管理

完整管理所有Claude Code会话，支持会话恢复、导出、标签分类和分析统计

🔍

Git集成

内置Git diff可视化工具，支持提交历史查看、分支管理和代码审查

📄

文件预览

在浏览器中直接预览代码文件，支持语法高亮、文件管理和版本比较

⏰

定时任务

支持定时执行Claude Code任务，提供可视化任务调度和监控界面

🌐

远程交互

支持远程访问和实时同步，提供安全认证和多用户权限管理

安装指南

多种安装方式，满足不同需求

🚀
Git克隆安装
安装命令 
# 克隆仓库
git clone https://github.com/d-kimuson/claude-code-viewer.git
cd claude-code-viewer

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env

# 启动服务
npm start

# 访问应用
# 浏览器打开: http://localhost:3000
🐳
Docker部署
Docker命令 
# 拉取镜像
docker pull d-kimuson/claude-code-viewer

# 运行容器
docker run -d \
  --name claude-viewer \
  -p 3000:3000 \
  -v /path/to/claude-projects:/app/projects \
  d-kimuson/claude-code-viewer

# 访问应用
# 浏览器打开: http://localhost:3000
⚡
一键部署
部署脚本 
# 下载部署脚本
curl -fsSL https://raw.githubusercontent.com/d-kimuson/claude-code-viewer/main/deploy.sh | bash

# 或手动下载执行
wget https://raw.githubusercontent.com/d-kimuson/claude-code-viewer/main/deploy.sh
chmod +x deploy.sh
./deploy.sh
