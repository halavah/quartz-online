# 用户指南

本指南将帮助您深入了解和使用 TechVerse 博客系统的各项功能。

## 🏠 首页功能

### 1. 网站信息展示

首页顶部显示：
- 网站名称（大标题）
- 网站描述（副标题）
- 固定在页面顶部，滚动时始终可见

### 2. 文章列表

- **单行展示**：每篇文章占一行，简洁明了
- **信息显示**：
  - 文章标题（主标题，较大字号）
  - 文章描述（副标题，灰色文字）
- **交互效果**：
  - 鼠标悬停时背景变灰
  - 标题变蓝色，提示可点击

### 3. 分页导航

- **每页显示**：10 篇文章
- **导航位置**：文章列表下方
- **智能显示**：
  - 始终显示第一页和最后一页
  - 当前页高亮显示
  - 显示前后相邻页码
  - 省略号表示更多页面

## 📄 文章详情页

### 1. 页面结构

```
┌─────────────────────────┐
│    返回按钮（顶部导航）   │
├─────────────────────────┤
│                         │
│    HTML 文章内容区域     │
│    （完全自定义）        │
│                         │
└─────────────────────────┘
```

### 2. 导航功能

- **返回按钮**：左上角，点击返回首页
- **固定位置**：始终在页面顶部，方便返回

### 3. 文章内容

- **完全由 HTML 控制**：系统不限制内容格式和样式
- **宽度自适应**：可以使用全宽，不受博客框架限制
- **自定义样式**：支持在 HTML 中内嵌 `<style>` 标签

## 🔗 URL 路由结构

### 首页
- URL: `/`
- 示例: `https://yourblog.com/`

### 文章详情页
- URL: `/article/[分类]/[文章名]`
- 基于文章文件路径自动生成

示例：
- 文件: `public/tech/nextjs-15-features.html`
- URL: `https://yourblog.com/article/tech/nextjs-15-features`

## 📝 文章管理

### 添加新文章

1. **创建 HTML 文件**：
   - 在 `public/` 下的任意目录创建 `.html` 文件
   - 文件名使用英文，避免特殊字符

2. **更新配置**：
   - 编辑 `data/articles.json`
   - 添加文章元数据

3. **重新构建**：
   - 运行 `npm run build`

### 文章元数据说明

```json
{
  "title": "文章标题",           // 必填，显示在列表和详情页
  "description": "文章简介",     // 必填，显示在列表页
  "htmlFile": "path/file.html"  // 必填，相对于 public 的路径
}
```

## 🎨 主题定制

### 修改网站信息

编辑 `data/articles.json`：

```json
{
  "config": {
    "siteName": "您的博客名称",
    "siteDescription": "您的博客描述"
  }
}
```

### 自定义颜色方案

编辑 `tailwind.config.ts`，添加自定义颜色：

```typescript
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: {
          50: '#eff6ff',
          500: '#3b82f6',
          900: '#1e3a8a'
        }
      }
    }
  }
}
```

### 修改字体

编辑 `app/globals.css`：

```css
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@300;400;600;700&display=swap');

html {
  font-family: 'Inter', -apple-system, BlinkMacSystemFont, sans-serif;
}
```

## 🔍 使用技巧

### 1. 文章分类

通过文件路径组织文章：

```
public/
├── frontend/               # 前端相关
│   ├── react.html
│   └── vue.html
├── backend/               # 后端相关
│   ├── nodejs.html
│   └── python.html
└── general/               # 通用文章
    └── best-practices.html
```

### 2. HTML 文章编写建议

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>文章标题</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <style>
        /* 文章特定样式 */
        .article-container {
            max-width: 900px;
            margin: 0 auto;
            padding: 40px;
        }
        .article-header {
            background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
            color: white;
            padding: 60px 40px;
        }
    </style>
</head>
<body>
    <div class="article-container">
        <!-- 文章内容 -->
    </div>
</body>
</html>
```

### 3. 性能优化

- 图片优化：使用 WebP 格式
- CSS 优化：避免重复样式
- JavaScript：只在需要时使用

在 `app/layout.tsx` 中添加：

```typescript
export default function RootLayout({ children }) {
  return (
    <html>
      <head>
        {/* Google Analytics */}
        <script async src="https://www.googletagmanager.com/gtag/js?id=GA_ID" />
        <script dangerouslySetInnerHTML={{
          __html: `
            window.dataLayer = window.dataLayer || [];
            function gtag(){dataLayer.push(arguments);}

            gtag('config', 'GA_ID');
          `

      </head>
      <body>{children}</body>
    </html>
  );
```

## ❓ 常见问题

### Q: 如何修改网站图标？
A: 在 `public/` 目录下放置 `favicon.ico` 文件。

### Q: 文章支持 Markdown 吗？
A: 目前系统只支持 HTML。您可以使用工具将 Markdown 转换为 HTML。

### Q: 如何添加代码高亮？
A: 在 HTML 文章中引入 Prism.js 或 highlight.js。

### Q: 如何添加评论功能？
A: 可以集成第三方评论服务，如 Disqus 或 Gitalk。
