    <h1>TypeScript 高级类型技巧</h1>

    <p>TypeScript 的类型系统非常强大，掌握高级类型技巧可以让你的代码更加健壮和可维护。本文将介绍几个实用的高级类型技巧。</p>

    <h2>1. 条件类型 (Conditional Types)</h2>
    <p>条件类型允许我们根据类型关系来选择类型：</p>
    <pre><code>// 基础条件类型
type IsString&lt;T&gt; = T extends string ? true : false;

// 使用示例
type Test1 = IsString&lt;string&gt;  // true
type Test2 = IsString&lt;number&gt;  // false

// 实际应用：提取数组元素类型
type ArrayElement&lt;T&gt; = T extends (infer U)[] ? U : never;

type Numbers = ArrayElement&lt;number[]&gt;  // number</code></pre>

    <h2>2. 映射类型 (Mapped Types)</h2>
    <p>映射类型可以创建新的类型，基于现有类型的结构：</p>
    <pre><code>// 将所有属性变为可选
type Partial&lt;T&gt; = {
  [P in keyof T]?: T[P];
};

// 将所有属性变为只读
type Readonly&lt;T&gt; = {
  readonly [P in keyof T]: T[P];
};

// 实际应用：创建更新类型
type Update&lt;T&gt; = Partial&lt;Pick&lt;T, Exclude&lt;keyof T, 'id'&gt;&gt;&gt;;

interface User {
  id: number;
  name: string;
  email: string;
}

type UserUpdate = Update&lt;User&gt;;
// 等价于: { name?: string; email?: string; }</code></pre>

    <h2>3. 模板字面量类型 (Template Literal Types)</h2>
    <p>TypeScript 4.1+ 支持模板字面量类型：</p>
    <pre><code>type EventName = `on${Capitalize&lt;string&gt;}`;

// 创建事件处理器类型
type EventHandlers&lt;T&gt; = {
  [K in keyof T as K extends string
    ? `on${Capitalize&lt;K&gt;}`
    : never]: T[K] extends (...args: any[]) => any
      ? T[K]
      : (event: any) => void;
};

// 使用示例
interface Events {
  click: (e: MouseEvent) => void;
  scroll: (e: Event) => void;
}

type Handlers = EventHandlers&lt;Events&gt;;
// 等价于: { onClick: (e: MouseEvent) => void; onScroll: (e: Event) => void; }</code></pre>

    <h2>4. 递归类型</h2>
    <div class="example">
        <strong>示例：深度只读</strong>
        <pre><code>type DeepReadonly&lt;T&gt; = {
  readonly [P in keyof T]: T[P] extends object
    ? DeepReadonly&lt;T[P]&gt;
    : T[P];
};

interface Nested {
  user: {
    name: string;
    address: {
      street: string;
    };
  };
}

type DeepNested = DeepReadonly&lt;Nested&gt;;
// 所有嵌套属性都变为只读</code></pre>
    </div>

    <h2>5. 工具类型组合</h2>
    <div class="tip">
        <strong>技巧</strong>：组合多个工具类型创建复杂类型
    </div>
    <pre><code>// 提取函数返回类型
type ReturnType&lt;T&gt; = T extends (...args: any[]) => infer R ? R : never;

// 提取 Promise 的值类型
type UnpackPromise&lt;T&gt; = T extends Promise&lt;infer U&gt; ? U : T;

// 组合使用：异步函数的返回类型
type AsyncReturnType&lt;T&gt; = UnpackPromise&lt;ReturnType&lt;T&gt;&gt;;

async function fetchUser(): Promise&lt;{ id: number; name: string }&gt; {
  return { id: 1, name: 'John' };
}

type UserType = AsyncReturnType&lt;typeof fetchUser&gt;;
// { id: number; name: string }</code></pre>

    <h2>6. 类型守卫和类型谓词</h2>
    <pre><code>// 类型谓词函数
function isString(value: unknown): value is string {
  return typeof value === 'string';
}

// 使用
function processValue(value: unknown) {
  if (isString(value)) {
    // TypeScript 知道这里 value 是 string
    console.log(value.toUpperCase());
  }
}

// 更复杂的类型守卫
function isUser(obj: any): obj is User {
  return obj && typeof obj.id === 'number' && typeof obj.name === 'string';
}</code></pre>

    <h2>7. 使用 infer 提取类型</h2>
    <pre><code>// 从函数参数提取类型
type FirstArg&lt;T&gt; = T extends (first: infer F, ...args: any[]) => any ? F : never;

type Test1 = FirstArg&lt;(name: string, age: number) => void&gt;;  // string

// 从 Promise 提取类型
type Awaited&lt;T&gt; = T extends Promise&lt;infer U&gt; ? U : T;

type Test2 = Awaited&lt;Promise&lt;string[]&gt;&gt;;  // string[]</code></pre>

    <h2>8. Brand Types</h2>
    <div class="tip">
        <strong>用途</strong>：创建不能与其他类型互换的唯一类型
    </div>
    <pre><code>interface Brand&lt;T&gt; {
  __type: T;
  __brand: unique symbol;
}

type BrandId = Brand&lt;'ID'&gt;;

function createId(id: number): BrandId {
  return id as any;
}

// 防止误用
function processId(id: BrandId) {
  console.log(id);
}

// processId(123); // 错误！
processId(createId(123)); // 正确</code></pre>

    <h2>总结</h2>
    <p>TypeScript 的高级类型系统提供了强大的类型操作能力。通过合理使用这些技巧，你可以：</p>
    <ul>
        <li>编写类型安全的代码</li>
        <li>提供更好的 IDE 支持</li>
        <li>减少运行时错误</li>
        <li>提高代码的可维护性</li>
    </ul>
