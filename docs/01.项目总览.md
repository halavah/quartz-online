# TechVerse 博客系统文档

欢迎使用 TechVerse 博客系统！这是一个现代化的技术博客平台，支持外部 HTML 文章挂载，配置简单，易于维护。

## 📚 文档目录

1. [快速开始](./04.快速开始.md) - 快速上手指南

3. [内容管理](./06.内容管理.md) - 文章发布和管理
4. [部署指南](./08.部署指南.md) - 部署到生产环境
5. [配置参考](./07.配置参考.md) - 系统配置说明
6. [常见问题](./09.常见问题.md) - 常见问题解答

## 🚀 核心特性

- **外部 HTML 挂载** - 文章内容完全由 HTML 文件控制
- **JSON 配置管理** - 所有文章元数据通过配置文件管理
- **静态站点生成** - 构建后生成纯静态文件，部署简单
- **现代化设计** - 简洁、大气、专业的视觉体验
- **分页导航** - 支持分页浏览，每页显示 10 篇文章
- **响应式布局** - 完美适配桌面和移动端
- **固定头部** - 导航始终可见，提升用户体验

## 🛠️ 技术栈

- **框架**: Next.js 15+ (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **构建工具**: Webpack (Next.js 内置)
- **部署方式**: 静态站点生成 (SSG)

## 🎯 适用场景

- 产品更新日志
- 开源项目文档

## 💡 设计理念

1. **内容优先** - 专注于内容展示，减少不必要的干扰
2. **简洁优雅** - 清晰的视觉层次，舒适的阅读体验
3. **易于维护** - 配置集中，结构清晰
4. **性能优先** - 静态生成，加载速度快
5. **可扩展性** - 模块化设计，易于添加新功能
